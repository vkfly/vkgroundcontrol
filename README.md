# 微克地面站

*微克地面站* （VKGCS）是基于QGC（QGroundControl）开发的地面站软件，用于帮助用户快速开发出操作基于微克V10/V12飞控的无人机。

VKGCS使用QML用于界面开发，底层与飞控的通信部分闭源，以SDK的方式提供。SDK将飞控信息直接暴露给QML层，用户可以使用相关的属性来显示飞机信息，调用公开的方法来控制无人机，也可以利用信号-槽的方式做更加灵活的处理。
